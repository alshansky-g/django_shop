# Django Shop Project

## Project Overview

Django Shop - это веб-приложение интернет-магазина, разработанное на фреймворке Django. Проект включает в себя функциональность для управления товарами, корзиной покупок, заказами и пользователями. Приложение написано на Python 3.14+ и использует PostgreSQL в качестве базы данных.

### Архитектура проекта

Проект состоит из следующих основных приложений:
- `main` - основная страница и навигация
- `goods` - каталог товаров
- `users` - управление пользователями, регистрация и аутентификация
- `carts` - корзина покупок
- `orders` - заказы пользователей
- `app` - основная конфигурация Django

### Основные зависимости

- Django 5.2.8
- PostgreSQL (через psycopg)
- Pillow (для работы с изображениями)
- django-debug-toolbar (для отладки)
- djlint (для форматирования HTML)

## Building and Running

Для запуска проекта:

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
   или если используется uv:
   ```bash
   uv sync
   ```

2. Настройте переменные окружения в файле `.env`:
   ```
   NAME=your_db_name
   USER=your_db_user
   PASSWORD=your_db_password
   HOST=your_db_host
   PORT=your_db_port
   ```

3. Выполните миграции:
   ```bash
   python manage.py migrate
   ```

4. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```

## Development Conventions

- Код форматируется с использованием Ruff (настройки в pyproject.toml)
- Для проверки качества кода используется набор линтеров, включая Django-специфичные правила (DJ)
- Язык интерфейса по умолчанию - русский (LANGUAGE_CODE = 'ru')
- Пользовательская модель переопределена (AUTH_USER_MODEL = 'users.User')
- Для работы с изображениями используется Pillow

## Особенности реализации

- Пользовательская модель наследуется от AbstractUser с добавлением полей для аватара, номера телефона и количества заказов
- Используется классическая MVC-архитектура Django с разделением на модели, представления и шаблоны
- Поддержка загрузки и хранения медиафайлов (изображений товаров и аватаров)
- Интеграция с debug_toolbar для отладки производительности
